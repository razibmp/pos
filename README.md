# 🎮 The Hobby Center — Dashboard v2

## Requirements

Install **Node.js** (free): https://nodejs.org
→ Download the **LTS** version and install it. That's all you need.

---

## How to Run (Windows)

1. Extract this ZIP folder anywhere on your PC
2. Double-click **start.bat**
3. Wait ~1 minute for it to install and build
4. Open your browser: **http://localhost:3001**

## How to Run (Mac / Linux)

1. Extract the ZIP
2. Open Terminal inside the folder
3. Run:
   ```
   chmod +x start.sh && ./start.sh
   ```
4. Open your browser: **http://localhost:3001**

## To stop the server

Press **Ctrl + C** in the terminal window.

## To start again next time

Just double-click **start.bat** again (Windows),
or run `cd server && node index.js` (Mac/Linux — no need to rebuild).

## Your data

All data is saved in `server/data/hobby-center.db`
This file keeps your data safe. Back it up anytime by copying it.
